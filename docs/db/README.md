# Configuration and DB formatting

## Files and General Purposes
- Config.yml
  - API cred
  - Users & Playlist URLs
  - Other Config information
- db.json
  - Stores info of generated playlists
- plists.json
  - All user playlists
  - Names and associations to playlists
  - Plist data
  - Single playlists
- gen_plist/
  - Folder holding all data of playlists
